Data

The data come from the NWPU-RESISC45 satellite dataset . As described in the Overview section, you are provided with a sub selection of 13 classes of this dataset, each containing 700 examples of different landscapes. Unlike the challenge on raw data, here you will work on a new representation of the original images consisting of 4096 features.

You must create a model which is able to predict the label of the preprocessed image given as input.

Go to the FILES tab to download the data and the starting kit. The starting kit contains a very small subset of the data for debug purposes. To prepare a challenge submission, you need to use the large dataset that you download separately. For the dataset, you might need to split a big test set to do benchmarking!
