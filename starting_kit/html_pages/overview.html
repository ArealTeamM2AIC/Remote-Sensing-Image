Solve an Aerial image recognition problem

Brought to you by the ChaLearn

The data set contains 9100 color images of size 128 x 128, images of 13 class, each having 700 images.
The classes are all a kind of landscapes or natural contrusction, and more specifically :
beach, chaparral, cloud, desert, forest, island, lake, meadow, mountain, river, sea, snowberg, wetland.

The goal of this challenge will be to classify each image and assign to it the correct label.
